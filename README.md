# Ayush Green Plant 🌿

## Project Overview

Ayush Green Plant is a web application designed to help users explore and discover different types of plants. The platform provides information about plants, their benefits, and promotes green living by encouraging people to learn more about nature.

## Features

* 🌱 Browse different plant categories
* 📖 View plant details and descriptions
* 🔍 Search and explore plants easily
* 💻 Responsive and modern UI

## Technologies Used

This project was built using:

* **React**
* **TypeScript**
* **Vite**
* **Tailwind CSS**
* **shadcn-ui**

## Installation & Setup

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone https://github.com/swaathy2007/auyus-green-main-1-.git
```

### 2. Navigate to the project folder

```bash
cd auyus-green-main-1-
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```

The application will start locally and you can view it in your browser.
